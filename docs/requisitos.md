# Requisitos Funcionais

## RF01 - Cadastrar cinemas
O sistema deve permitir o cadastro de cinemas com informações básicas como nome, endereço e capacidade.

## RF02 - Cadastrar filmes
O sistema deve permitir o cadastro de filmes contendo título, gênero, diretor e duração.

## RF03 - Cadastrar sessões
O sistema deve permitir o cadastro de sessões associando filmes e cinemas.

## RF04 - Registrar público da sessão
O sistema deve permitir registrar a quantidade de espectadores presentes em cada sessão.

## RF05 - Consultar filmes em cartaz
O sistema deve permitir visualizar os filmes disponíveis em exibição.

## RF06 - Consultar sessões por cinema
O sistema deve permitir consultar as sessões disponíveis em cada cinema.

## RF07 - Consultar total de público por filme
O sistema deve permitir visualizar o total de espectadores de cada filme.
